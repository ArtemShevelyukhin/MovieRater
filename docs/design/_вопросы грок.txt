Вариант 5. Dependency на уровне роутера

Если много эндпоинтов используют БД:

router = APIRouter(dependencies=[Depends(get_db)])

@router.post("/register")
def register(telegram_token: str, db: Session = Depends(get_db)):
    ...


Или даже так:

@router.post("/register")
def register(
    telegram_token: str,
    db: Session = Depends()
):


(если get_db — dependency по умолчанию)

✔️ уменьшает дублирование
✔️ чистый код
